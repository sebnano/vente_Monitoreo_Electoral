## Configuración para desarrollo

### Requerimientos

-   Visual Studio 2022 o Visual Studio for Mac con los SDK de .NET MAUI.
-   SDK de Android e iOS actualizados.
-   SDK .NET actualizados.
-   Emulador Android/iOS o dispositivo físico.
